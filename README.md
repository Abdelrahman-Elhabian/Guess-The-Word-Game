A fun and interactive **Wordle-inspired** guessing game built with **vanilla JavaScript, HTML, and CSS**.  
Guess the hidden 6-letter word in 6 tries. Get color feedback just like the original Wordle!

## Features

- 6-letter word to guess in 6 attempts
- Color-coded feedback:
  - Orange: Correct letter, correct position
  - Green: Correct letter, wrong position
  - Dark gray: Letter not in the word
- Hint system (2 hints available)
- Auto-focus next input field when typing
- Arrow key navigation + Backspace support
- Responsive and clean UI
- Fully functional with no external libraries

## How to Play

1. A random 6-letter word is chosen from a predefined list.
2. Type your guess and press "Check Word".
3. Use the color clues to refine your next guess.
4. Use "Hints" if you're stuck â€” it will fill in one correct letter.
5. Win by guessing the word in 6 tries or less!

> 
6. Bonus message if you win without using hints!
